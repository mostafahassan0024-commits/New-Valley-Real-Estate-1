# تصميم تطبيق عقارات الوادي الجديد

## نظرة عامة
تطبيق موبايل متكامل لعرض وإدارة العقارات في محافظة الوادي الجديد (شقق سكنية، أراضي زراعية وسكنية) مع نظام تواصل داخلي وواتساب.

## الهوية البصرية - مستوحاة من بيئة الوادي الجديد
- **الألوان الرئيسية:**
  - الأخضر الصحراوي (#2D5016) - يمثل الزراعة والواحات
  - الذهبي الرملي (#D4A574) - يمثل الصحراء والكثبان الرملية
  - الأزرق الواحي (#0F7EA4) - يمثل المياه والآبار
  - البني الترابي (#8B6F47) - يمثل الأرض والتراث
- **الخطوط:** خطوط عربية واضحة تدعم القراءة السهلة
- **الأيقونات:** بسيطة وواضحة تعكس الطابع المحلي

## قائمة الشاشات

### 1. شاشة الترحيب (Splash)
- عرض شعار التطبيق مع خلفية مستوحاة من الوادي الجديد
- انتقال تلقائي لشاشة تسجيل الدخول أو الرئيسية

### 2. شاشة تسجيل الدخول/التسجيل (Auth)
- نموذج تسجيل الدخول (رقم الهاتف/البريد الإلكتروني + كلمة المرور)
- زر "تسجيل حساب جديد"
- خيار "نسيت كلمة المرور"
- تسجيل الدخول عبر OAuth (اختياري)

### 3. الشاشة الرئيسية (Home)
**المحتوى الأساسي:**
- شريط بحث في الأعلى (بحث بالاسم، الموقع، السعر)
- فلاتر سريعة (نوع العقار: شقق، أراضي زراعية، أراضي سكنية)
- قائمة العقارات المتاحة (بطاقات عرض)
  - صورة العقار الرئيسية
  - العنوان والموقع
  - السعر
  - نوع العرض (بيع/إيجار)
  - أيقونة قلب للمفضلة

**الوظائف:**
- الضغط على بطاقة عقار → الانتقال لشاشة التفاصيل
- سحب للأسفل لتحديث القائمة
- تحميل تلقائي عند التمرير

### 4. شاشة تفاصيل العقار (Property Details)
**المحتوى:**
- معرض صور قابل للتمرير (Swipeable Gallery)
- معلومات العقار:
  - العنوان والموقع (مع خريطة صغيرة)
  - السعر ونوع العرض (بيع/إيجار)
  - المساحة (بالمتر المربع)
  - نوع العقار (شقة، أرض زراعية، أرض سكنية)
  - عدد الغرف (للشقق)
  - الوصف التفصيلي
  - تاريخ النشر
- معلومات المالك/المعلن:
  - الاسم
  - رقم الهاتف
- أزرار الإجراءات:
  - زر "تواصل عبر الواتساب" (أخضر)
  - زر "مراسلة داخل التطبيق" (أزرق)
  - زر "إضافة للمفضلة" (قلب)
  - زر "مشاركة العقار"

**الوظائف:**
- عرض الصور بملء الشاشة عند الضغط
- فتح الخريطة بموقع العقار
- فتح واتساب مع رسالة جاهزة
- فتح شاشة المحادثة الداخلية

### 5. شاشة البحث والفلترة (Search & Filter)
**المحتوى:**
- حقل البحث النصي
- فلاتر متقدمة:
  - نوع العقار (شقق، أراضي زراعية، أراضي سكنية)
  - نوع العرض (بيع، إيجار)
  - نطاق السعر (من - إلى)
  - نطاق المساحة (من - إلى)
  - الموقع (اختيار من قائمة المناطق)
  - عدد الغرف (للشقق)
- زر "تطبيق الفلاتر"
- زر "إعادة تعيين"

### 6. شاشة المفضلة (Favorites)
**المحتوى:**
- قائمة العقارات المحفوظة
- نفس تصميم بطاقات الشاشة الرئيسية
- رسالة "لا توجد عقارات محفوظة" عند الفراغ

**الوظائف:**
- إزالة من المفضلة بسحب البطاقة
- الضغط للانتقال لتفاصيل العقار

### 7. شاشة المحادثات (Messages)
**المحتوى:**
- قائمة المحادثات النشطة
- كل محادثة تعرض:
  - صورة المستخدم الآخر
  - الاسم
  - آخر رسالة
  - التوقيت
  - عدد الرسائل غير المقروءة
- رسالة "لا توجد محادثات" عند الفراغ

**الوظائف:**
- الضغط على محادثة → فتح شاشة الدردشة

### 8. شاشة الدردشة (Chat)
**المحتوى:**
- رأس الشاشة: صورة واسم المستخدم الآخر
- منطقة الرسائل:
  - رسائل المستخدم (جهة اليمين، لون أزرق)
  - رسائل الطرف الآخر (جهة اليسار، لون رمادي)
  - التوقيت أسفل كل رسالة
- حقل إدخال الرسالة في الأسفل
- زر إرسال

**الوظائف:**
- إرسال رسائل نصية
- التمرير التلقائي لآخر رسالة
- إشعارات الرسائل الجديدة

### 9. شاشة إضافة عقار (Add Property)
**المحتوى:**
- نموذج إدخال بيانات العقار:
  - العنوان
  - نوع العقار (اختيار من قائمة)
  - نوع العرض (بيع/إيجار)
  - السعر
  - المساحة
  - عدد الغرف (للشقق)
  - الوصف
  - الموقع (اختيار من خريطة أو قائمة)
  - تحميل الصور (حتى 10 صور)
- زر "نشر العقار"

**الوظائف:**
- التحقق من صحة البيانات
- رفع الصور
- حفظ العقار في قاعدة البيانات

### 10. شاشة عقاراتي (My Properties)
**المحتوى:**
- قائمة العقارات المنشورة من قبل المستخدم
- نفس تصميم بطاقات الشاشة الرئيسية
- أيقونات إضافية:
  - تعديل
  - حذف
  - إخفاء/إظهار

**الوظائف:**
- تعديل بيانات العقار
- حذف العقار
- إخفاء العقار مؤقتاً

### 11. شاشة الملف الشخصي (Profile)
**المحتوى:**
- صورة المستخدم
- الاسم
- رقم الهاتف
- البريد الإلكتروني
- إحصائيات:
  - عدد العقارات المنشورة
  - عدد العقارات المفضلة
  - عدد المحادثات
- زر "تعديل الملف الشخصي"
- زر "الإعدادات"
- زر "تسجيل الخروج"

### 12. شاشة الإعدادات (Settings)
**المحتوى:**
- الإشعارات (تفعيل/تعطيل)
- اللغة (عربي/إنجليزي)
- الوضع الليلي
- الخصوصية
- الشروط والأحكام
- من نحن
- تواصل معنا

## التنقل الرئيسي (Tab Bar)
- **الرئيسية** (Home) - أيقونة منزل
- **البحث** (Search) - أيقونة بحث
- **إضافة عقار** (Add) - أيقونة زائد (بارزة)
- **المحادثات** (Messages) - أيقونة رسالة (مع عداد)
- **الملف الشخصي** (Profile) - أيقونة مستخدم

## سيناريوهات الاستخدام الرئيسية

### سيناريو 1: البحث عن عقار
1. المستخدم يفتح التطبيق → الشاشة الرئيسية
2. يستخدم شريط البحث أو الفلاتر
3. يتصفح النتائج
4. يضغط على عقار → شاشة التفاصيل
5. يشاهد الصور والتفاصيل
6. يضغط "تواصل عبر الواتساب" → يفتح واتساب مع رسالة جاهزة

### سيناريو 2: إضافة عقار للبيع
1. المستخدم يضغط على تبويب "إضافة عقار"
2. يملأ النموذج (العنوان، النوع، السعر، الصور، إلخ)
3. يضغط "نشر العقار"
4. يظهر العقار في "عقاراتي" وفي الشاشة الرئيسية

### سيناريو 3: التواصل الداخلي
1. المستخدم في شاشة تفاصيل عقار
2. يضغط "مراسلة داخل التطبيق"
3. يفتح شاشة الدردشة
4. يكتب رسالة ويرسلها
5. يتلقى رد من المالك
6. تظهر المحادثة في تبويب "المحادثات"

### سيناريو 4: حفظ عقار في المفضلة
1. المستخدم يتصفح العقارات
2. يضغط على أيقونة القلب في بطاقة عقار
3. يُحفظ العقار في "المفضلة"
4. يمكن الوصول إليه لاحقاً من تبويب "المفضلة"

## اختيارات الألوان للتطبيق
- **Primary (اللون الأساسي):** #0F7EA4 (الأزرق الواحي)
- **Secondary (اللون الثانوي):** #2D5016 (الأخضر الصحراوي)
- **Accent (لون التمييز):** #D4A574 (الذهبي الرملي)
- **Background (الخلفية):** #FFFFFF (فاتح) / #151718 (داكن)
- **Surface (الأسطح):** #F5F5F5 (فاتح) / #1E2022 (داكن)
- **Text Primary:** #11181C (فاتح) / #ECEDEE (داكن)
- **Text Muted:** #687076 (فاتح) / #9BA1A6 (داكن)
- **Success:** #22C55E (أخضر)
- **Warning:** #F59E0B (برتقالي)
- **Error:** #EF4444 (أحمر)

## ملاحظات تصميمية
- التصميم يتبع معايير Apple Human Interface Guidelines
- الواجهة مصممة للاستخدام بيد واحدة (Portrait 9:16)
- الأزرار الرئيسية في متناول الإبهام
- استخدام الحركات البسيطة (Swipe, Tap, Long Press)
- التركيز على الصور عالية الجودة للعقارات
- دعم الوضع الليلي بالكامل
- دعم اللغة العربية بشكل كامل (RTL)
